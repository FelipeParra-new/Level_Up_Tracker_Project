# Level_Up_Tracker_Project
Un tracker de entrenamiento para registrar el volumen total de ejercicio y optimizar las recomendaciones de peso de forma inteligente.
Mis registros rastrean el inicio de la aplicación, los errores al ingresar datos y los cálculos exitosos de los ejercicios. Elegí esto para ayudar a identificar rápidamente los errores del usuario y mantener un historial de las sesiones completadas sin que el programa se cierre de golpe.
